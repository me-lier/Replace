How to get your Render API Key:

  1. Go to Render API Keys Page
  2. Click "New API Key"
  3. Give it a name (e.g., "Chatbot Deployer")
  4. Copy the generated key (it starts with "rk_")
  5. Paste it here

How to find your Render Owner ID:

  1. Go to Render Dashboard
  2. Click your profile picture in the top right
  3. Select "Account Settings"
  4. Your Owner ID will be in the URL: https://dashboard.render.com/account/usr-xxxxx
  5. Copy the usr-xxxxx part
